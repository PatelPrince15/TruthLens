Welcome to TruthLens

TruthLens is a simulated text lie detector built entirely with frontend technologies. It analyzes your statements using simple rule-based logic, keyword detection, and pattern recognition to estimate whether a statement is “Truth Likely” or “Lie Likely”. This project is for educational and fun purposes only — not real lie detection.
