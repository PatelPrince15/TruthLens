TruthLens – How True Is Your Statement?

What is TruthLens?
TruthLens is a lightweight web app that gives your statements a **truth score**.  
Instead of a boring true/false answer, it estimates **how truthful something sounds** in percentage form.

 Why This Project?
Because truth isn’t always black or white.  
This project explores how simple logic can simulate **human-like judgment** using frontend technologies.

 Features
- Enter any statement  
- Get a truth percentage
- Instant result  
- Clean and distraction-free UI  

 Built With
- HTML  
- CSS  
- JavaScript  

How It Works
Type → Click → Analyze → Get Score  

 Disclaimer
TruthLens is a **simulation project** and does not use real AI.  
Results are generated using basic logic for demonstration purposes only.
